# 📊 Data Science Portfolio

Welcome! This repository contains my end-to-end Machine Learning projects, focusing on solving real-world business problems through data-driven insights.

## 🚀 Projects Overview

### 1. Telco Customer Churn Prediction (Classification)
- **Objective:** Predict which customers are likely to leave the telecom service to help in retention strategies.
- **Dataset:** 20,000+ rows of customer behavior data.
- **Key Implementation:** - **Algorithm:** Logistic Regression.
  - **Threshold Optimization:** Adjusted the classification threshold to **0.4** to maximize Recall, ensuring critical churn cases are not missed.
  - **Data Cleaning:** Handled null values using **Median Imputation** to maintain data integrity against outliers.
  - **Visualizations:** Sigmoid curve analysis and Confusion Matrix.

### 2. Linear & Multiple Regression Analysis
- **Objective:** Predicting numerical outcomes and understanding feature correlations.
- **Key Implementation:** - Exploratory Data Analysis (EDA) to identify multi-collinearity.
  - Feature scaling and preprocessing for better model convergence.
